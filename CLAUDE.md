# TRÓPICO EVENTOS — Brand Guidelines

## Identidade Visual

### Cores
- Background principal: #F2F0EB (Areia)
- Background escuro / CTA: #3A3A38 (Carvão)
- Preto da marca: #1A1A18
- Acento único: #8A7F5A (Dourado Fosco)
- Acento claro: #C4B89A
- Texto corpo: #6B6B68
- Bordas / divisores: #E0DDD6
- Logotipo fantasma: #B8B6AE (8% opacidade)

### Tipografia
- Família: DM Sans (Google Fonts) — única família, sem mix
- Display/Logo: 700 bold, uppercase, tracking -0.01em
- Heading 1: 700 bold, 36–48px
- Labels/Nav: 500 medium, uppercase, tracking +0.12em
- Corpo: 300 light, 15–16px, line-height 1.7
- Caption: 500 medium, 11px, uppercase, tracking +0.10em

### Logo
- Arquivo principal: /assets/logo/LOGO.png (PNG com fundo transparente, tinta preta)
- Composição: wordmark tipográfico vertical "TRÓPICO" — tipografia display customizada, não recriar
- Versão fundo claro: usar LOGO.png diretamente
- Versão fundo escuro: aplicar CSS `filter: invert(1)` ou usar arquivo LOGO-white.png separado
- Proporção: vertical, aproximadamente 0.6:1 (largura:altura)
- Área de respiro mínima: equivalente à largura da letra "P" em todos os lados
- Nunca distorcer proporções, recortar ou recriar tipograficamente
- Variação secundária aceitável: wordmark horizontal "TRÓPICO" em DM Sans 700 uppercase, tracking +0.14em — apenas quando o logo principal não couber no layout

### Elementos visuais
- Símbolo: três círculos sobrepostos, stroke fino #8A7F5A, nunca preenchido
- Logotipo fantasma: "TRÓPICO" como elemento de background, opacidade ~8%
- Linhas finas: 0.5–1px, usadas como separadores assimétricos
- Três pontos: marcador de canto, cria enquadramento
- Fotos de pessoas: sempre em P&B, crop circular, borda dourada
- Fotos de eventos: coloridas, servem como contraste intencional com o layout neutro

## Tom de Voz

### Pilares
- Poético sem ser vago
- Confiante sem arrogância
- Humano e direto
- O detalhe como diferencial

### Taglines da marca
- "Para cruzar a fronteira do óbvio."
- "Prepare-se para cruzar a linha."
- "Pelo arrepio do novo. Pela busca do que permanece."

### Evitar
- Jargão corporativo ("soluções", "expertise", "metodologias")
- Superlativos genéricos ("o melhor", "líder de mercado")
- Linguagem passiva e burocrática
- CTAs frios ("Entre em contato para solicitar orçamento")

## Regras de Design

### Sempre
- Fundo neutro como moldura para conteúdo intenso
- Espaço em branco generoso — o vazio é parte do design
- Hierarquia clara via tipografia, não via cor
- Animações de entrada sutis (fade + translate, sem bounce)

### Nunca
- Gradientes no layout institucional
- Múltiplas famílias tipográficas
- Cores saturadas fora de fotos de eventos
- Sombras pesadas (box-shadow máximo: 0 2px 8px rgba(0,0,0,0.06))
- Stock photos
- Ícones genéricos de clipart
- Recriar o logo tipograficamente — sempre usar o arquivo PNG
